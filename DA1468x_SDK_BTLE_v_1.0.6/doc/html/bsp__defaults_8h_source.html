88 /* Assuming that the frequency of the external digital clock is 32000Hz,
89  * period: 1/32000 = 31.25usec
90  * Timer1 wrap-around time: 65536 * 31.25 = 2.048sec
91  * With a prescaler equal to 3, the wrap around time becomes: (1 + 3) * 2.048 = ~8sec.
92  * If the clock frequency is too slow, i.e. 16000Hz then
93  * period: 1/16000 = 62.5usec
94  * Timer1 wrap-around time = 65536 * 62.5 = 4.096sec
95  * and the prescaler should be 1 so that: (1 + 1) * 4.096 = ~8sec.
96  * If the clock frequency is too high, i.e. 125000Hz then
97  * period: 1/125000 = 8usec
98  * Timer1 wrap-around time = 65536 * 8 = 0.524sec
99  * and the prescaler should be 3 so that: (1 + 3) * 0.524 = ~2sec.
100  * The values MUST be defined in the custom_config_<>.h file!
101  */

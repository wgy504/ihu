bool irb_execute ( void *  cmd,
void **  rsp 
)

Execute IRB.

This creates IRB with given message, sends it to manager and waits for response. Buffer pointed by cmd is owned by manager after calling this function and should not be accessed. Buffer returned in rsp is owned by caller and should be freed there.

Parameters
[in]cmdcommand buffer
[out]rspresponse buffer
Returns
true if executed successfully, false otherwise

To execute the code via command line, do the following:
1. Edit the file jlink_download.bat to point to the correct installation path of the Segger tools in your PC.
2. Copy the files jlink_download.bat and jlink_download.bin to the Debug/ or Release/ directory.
3. Open a command shell, go to the Debug_RAM/ or Release_RAM/ directory and run jlink_download.bat.
